# Cara menambahkan gambar mockup

## Struktur folder

```
public/
├── hero-bg.jpg              ← foto background hero section
└── mockups/
    ├── vellum-01.jpg
    ├── vellum-02.jpg
    ├── vellum-03.jpg
    ├── struktur-01.jpg
    ├── struktur-02.jpg
    ├── struktur-03.jpg
    └── ... dst
```

## Spesifikasi gambar yang disarankan

| Gambar         | Ukuran ideal   | Rasio  | Format |
|----------------|----------------|--------|--------|
| hero-bg.jpg    | 2560 x 1440px  | 16:9   | JPG    |
| mockup utama   | 1920 x 820px   | 21:9   | JPG    |
| mockup thumb   | 800 x 500px    | 16:10  | JPG    |

## Tips foto mockup yang bagus

- Font dipakai di cover majalah / editorial spread
- Font dipakai di packaging produk (kosmetik, kopi, buku)
- Font dipakai di poster atau signage
- Cahaya natural, komposisi bersih
- Hitam putih atau desaturated — menjaga konsistensi palette

## Catatan

Selama file gambar belum ada, website otomatis menampilkan
placeholder SVG — tidak error, tetap terlihat profesional.
Begitu file JPG diletakkan dengan nama yang benar, langsung terganti.
